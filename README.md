# hobbies-book
Hobbi book with login system (MERN)

### [LIVE](https://big-bhavya.herokuapp.com/)
### [Hobbies Book](https://big-bhavya.herokuapp.com/)

## technology stack
- React
- Node.js
- MongoDB
- Express
- JWT
- Context

## Features
- Create account 
- Login with email and password
- Post Hobbies
- Set user profile with nice animation and design
- Lifetime accessible
